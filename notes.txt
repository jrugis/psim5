Parotid Simulation on Mahuika
=============================
cd /nesi/project/nesi00119/your_name
git clone https://github.com/jrugis/psim5.git


Prerequisites
-------------
(could be put in .bash_profile)
ml gimkl/2017a
ml Eigen/3.3.4
ml Python/2.7.14-gimkl-2017a


Build
-----
cd psim5/build
make
chmod 770 ../run/psim5


Run
---
cd psim5/run


Post
----
cd psim5/post
